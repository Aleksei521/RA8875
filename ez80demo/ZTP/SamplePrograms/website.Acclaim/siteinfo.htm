This Site - Hardware
This site is powered by a single eZ80 Internet Processor.
The hardware used is an eZ80Acclaim!™ Internet Processor. The following lists the basic components found on the eZ80Acclaim!™:
eZ80Acclaim!™ Internet Processor
Integrated 10/100Mbps Ethernet controller (eZ80F91)
10Mbps external MAC and PHY (eZ80F92 and eZ80F93)
1MB of external Flash
2 serial ports supporting PPP and background debugging
System and GPIO expander connectors


The eZ80Acclaim!™, when running as the Zilog embedded Web server demo, is a full TCP/IP compliant network device and can be plugged directly into a typical local Intranet. If a DHCP server on the Intranet is found during initial startup, the evluation board will automatically negotiate with the DHCP server and configure itself to run on the Intranet. If a DHCP server is not found, the default network configuration will be used.

One can also use an Ethernet cross-over cable to connect a standard PC Ethernet port to the eZ80Acclaim!™ directly. In this case the eZ80Acclaim!™ will operate with its default network settings. In this mode, the PC's Ethernet port must sit on the same subnet as the eZ80Acclaim!™ (or vise versa).

This Site - Software

This site is powered by a single eZ80 Internet Processor. The software components consist of the following:

Standard Web pages from MS FrontPage 2000
ZTP TCP/IP Software Suite with integrated RTOS
RTOS includes network interface drivers (Ethernet and serial modem)
Application Software implementing an embedded Webserver

These components are compiled and linked into a final output file in Intel hex format, using Zilog's Developer Studio and the eZ80 C-Compiler. The hex file is then downloaded into the eZ80Acclaim!™.
Multiple network sockets can be opened simultaneously. Maximum sockets are limited by system memory, and is controlled by the OS.
