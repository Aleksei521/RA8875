TCP/IP Protocol Suite
ZTP TCP/IP Software Suite includes everything you need to connect an appliance to a TCP/IP network, including the Internet. The supported physical network interfaces are Ethernet and serial modem (PPP). Device drivers for Ethernet and serial modem are included.
User software interfaces to the TCP/IP software using standard API calls. Lower level network functions are handled by the TCP/IP software with minimum user intervention. Because ZTP TCP/IP software is already integrated with a kernel for real-time applications, OS API calls are made available for end user application.

Supported Protocols:
IPv4 HTTP v1.1
TCP SMTP
UDP TELNET
ARP/ RARP SNMP
ICMP/ IGMP TFTP
BOOTP/ DHCP DNS
PPP TIMEP

 

OS Feature Summary:
Pre-emptive Multi-Tasking
Thread management (priorities, create, suspend, resume, kill, etc.)
Inter-process communication (messaging queues, semaphores)
Networking capabilities (Server functions, TCP APIs, UDP APIs, etc.)
Memory management


All network APIs and OS APIs are documented for easy integration into the user's application. The entire software suite is organized in a manner where unused function calls are excluded from the final project build. This arrangement provides most flexibility and smallest code yield.
